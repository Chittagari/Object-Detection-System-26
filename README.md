# AI Object Detection Radar System

##  Project Overview

The **AI Object Detection Radar System** is an intelligent real-time monitoring solution that combines computer vision and sensor technology to detect and identify nearby objects. The system uses a camera to capture live video and an ultrasonic sensor to measure distance. The captured data is processed using **TensorFlow.js AI object detection models**, which identify objects and display them on a radar-style graphical interface.

The system provides real-time visual detection, distance measurement, and voice alerts to notify users about detected objects. This project demonstrates the integration of **Artificial Intelligence, IoT hardware, and web technologies** to create an interactive and smart detection platform.

---

##  Key Features

* Real-time AI object detection using camera input
* Distance measurement using ultrasonic sensor
* Radar-style graphical user interface for visualization
* Voice alert system for detected objects
* Integration of AI models with IoT hardware
* Real-time browser-based processing

---

##  Algorithms / Models Used

* **COCO-SSD Object Detection Model** – Detects and classifies multiple objects in real-time.
* **Convolutional Neural Networks (CNN)** – Used within the AI model for visual feature extraction and object recognition.

Install the following before running the project:

* Arduino IDE
* Modern Web Browser (Chrome recommended)
* Webcam
* Arduino UNO
* Ultrasonic Sensor (HC-SR04)
